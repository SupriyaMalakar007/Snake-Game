
# 🐍 Hand Controlled Snake Game

A futuristic Snake Game controlled using real-time hand tracking with AI-powered gesture recognition.

Built with:
- Python
- OpenCV
- MediaPipe
- Pygame

Control the snake using your hand movements through your webcam.

---

# 🎮 Features

- Real-time hand tracking
- Webcam-based controls
- Fullscreen gameplay
- Smooth snake movement
- AI gesture direction detection
- Score system
- Restart support
- Game over detection
- Optimized performance

---

# 🛠️ Technologies Used

- Python
- OpenCV
- MediaPipe
- Pygame
